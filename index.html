About Large Scale Malware Classification

In the study, we focus to detect and classify malwares by using machine learning algorithms. Before apply machine learning algorithms, executables' behavioural profiles are obtained by the dynamic analysis.
The specific aims of the study are to:

  • Automatically extract runtime behaviour of the executable file.
  • Convert runtime behaviour to a proper form before applying machine learning algorithm.
  • Select appropriate features for runtime behaviour.
  • Detect and classify malware by applying machine learning algorithms on selected features.

The samples used for evaluations is provided, and you can download and use it for any purpose.
